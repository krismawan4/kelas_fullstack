## Apa yang Dipelajari

- **Struktur dasar HTML:** `<!DOCTYPE>`, `<html>`, `<head>`, `<body>`.
- **Semantic HTML5:** `<header>`, `<nav>`, `<main>`, `<section>`, `<article>`, `<footer>`.
- **Elemen teks:** `<h1>`-`<h6>`, `<p>`, `<strong>`, `<em>`, `<br>`.
- **Elemen media:** `<img>`, `<video>`, `<audio>`, `<iframe>`.
- **Form & Input:** `<form>`, `<input>`, `<select>`, `<textarea>`, `<button>`.
- **Link & Tabel:** `<a>`, `<table>`, `<tr>`, `<td>`, `<th>`.

## Contoh Kode

```html
<!DOCTYPE html>
<html lang="id">
<head>
  <meta charset="UTF-8">
  <meta name="viewport" content="width=device-width">
  <title>Halaman Pertamaku</title>
</head>
<body>
  <header>
    <nav>
      <a href="/">Home</a>
      <a href="/about">Tentang</a>
    </nav>
  </header>

  <main>
    <section>
      <h1>Halo, Dunia! 👋</h1>
      <p>Ini halaman web pertama saya.</p>
    </section>
  </main>

  <footer>
    <p>© 2025 Nama Saya</p>
  </footer>
</body>
</html>
```

## Latihan

1. Buat halaman HTML lengkap dengan `header`, `main`, dan `footer`.
2. Tambahkan navigasi dengan 3 link.
3. Buat form registrasi sederhana (nama, email, password, tombol submit).
4. Tambahkan tabel berisi jadwal pelajaran.
