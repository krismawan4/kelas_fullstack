## Apa yang Dipelajari

- Daftar akun di [figma.com](https://figma.com) (gratis untuk pelajar).
- **Interface Figma:** Toolbar, Canvas, Layers, Properties.
- **Frame & Group:** Cara mengorganisir elemen desain.
- **Components & Variants:** Membuat komponen yang bisa dipakai ulang.
- **Auto Layout:** Membuat layout yang fleksibel seperti CSS Flexbox.
- **Prototype:** Menambahkan interaksi dan animasi pada desain.

## Shortcut Figma yang Wajib Dikuasai

| Shortcut | Fungsi |
|---|---|
| `F` | Frame tool |
| `R` | Rectangle |
| `T` | Text |
| `V` | Move/Select |
| `K` | Scale |
| `Ctrl+G` | Group |
| `Ctrl+Alt+G` | Frame selection |
| `Ctrl+D` | Duplicate |
| `Ctrl+Z` | Undo |

## Latihan

1. Buat akun Figma gratis.
2. Buat frame iPhone 14 dan desain halaman login sederhana.
3. Gunakan Auto Layout untuk membuat tombol yang responsif.
4. Buat 2 komponen: Button Primary dan Button Secondary.
