## Apa yang Dipelajari

- **Laravel (Inertia.js / Livewire):** Menyatukan kehebatan backend PHP dengan frontend modern (Vue/React) tanpa perlu repot membangun API secara manual berkat Inertia.js.
- **Hono & Edge Computing:** Framework web *ultrafast* yang berjalan di Cloudflare Workers, Deno, atau Bun. Sangat cocok untuk *microservices* dan API dengan latensi super rendah.
- **Pemilihan Tech Stack:** Kapan harus menggunakan Laravel, kapan harus menggunakan Next.js, dan kapan menggunakan Hono.

## Latihan

1. Buat API Todo List dengan performa tinggi menggunakan **Hono** dan *Cloudflare D1* (SQLite).
2. Buat aplikasi Todo List menggunakan **Laravel + Inertia.js + Vue**, dan rasakan perbedaannya dibandingkan stack MEVN biasa!
