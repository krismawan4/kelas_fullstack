## Apa yang Dipelajari

- **Evolusi SPA ke SSR:** Mengatasi masalah SEO dan performa awal (*first contentful paint*) dari SPA konvensional.
- **Next.js (React) & Nuxt.js (Vue):** Framework "Fullstack" yang menyatukan backend (API routes) dan frontend dalam satu basis kode (Monolith Modern).
- **Server Actions & Server Components:** Mengakses database secara langsung dari komponen UI tanpa perlu menulis *endpoint* API terpisah.

## Latihan

1. Ubah aplikasi Todo List MEVN/MERN Anda menjadi aplikasi Nuxt.js atau Next.js.
2. Gunakan *Prisma ORM* untuk mengambil data Todo langsung dari database di dalam *Server Component*.
