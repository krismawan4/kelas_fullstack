## Apa yang Dipelajari

- **Keluarga Stack JavaScript:** 
  - **M**ongoDB (Database)
  - **E**xpress.js (Backend Framework)
  - **A**ngular / **V**ue.js / **R**eact (Frontend Framework)
  - **N**ode.js (Runtime Environment)
- **Arsitektur SPA (Single Page Application):** Mengonsumsi REST API dari Backend Express.js menggunakan Axios atau Fetch API.
- **Kelebihan & Kekurangan:** Semuanya ditulis dalam satu bahasa (JavaScript/TypeScript), namun manajemen *state* dan integrasi API harus dilakukan secara manual.

## Latihan

1. Buat REST API Todo List menggunakan Express.js dan MongoDB.
2. Buat antarmuka pengguna (Frontend) menggunakan Vue.js atau React yang terhubung dengan API tersebut.
