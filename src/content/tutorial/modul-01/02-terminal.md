## Apa yang Dipelajari

- `cd folder/` — Berpindah ke folder tertentu.
- `ls` atau `dir` — Melihat isi folder.
- `mkdir nama-folder` — Membuat folder baru.
- `touch nama-file.js` — Membuat file baru.
- `rm nama-file` — Menghapus file.
- **PATH:** Cara sistem operasi menemukan program.

## Contoh Kode

```bash
# Perintah terminal yang wajib dikuasai

# Navigasi direktori
cd Desktop
cd ..           # kembali ke folder atas

# Membuat proyek baru
mkdir proyek-web
cd proyek-web
touch index.html style.css script.js

# Melihat isi folder
ls -la          # detail dengan hidden file
```

## Latihan

1. Buka terminal dan navigasi ke folder Desktop.
2. Buat folder baru bernama `latihan-terminal`.
3. Di dalamnya, buat 3 file: `index.html`, `style.css`, `app.js`.
4. Tampilkan isi folder dengan `ls -la`.
