## Apa yang Dipelajari

- Download VS Code dari [code.visualstudio.com](https://code.visualstudio.com).
- **Extension wajib:** Prettier, ESLint, GitLens.
- **Extension Vue:** Volar (pengganti Vetur).
- **Keyboard shortcut penting:** `Ctrl+P`, `Ctrl+Shift+P`.
- **Settings:** Auto-save, format on save, font size.

## Extensions yang Wajib Dipasang

```text
1. Prettier - Code formatter
2. ESLint
3. Auto Close Tag
4. GitLens
5. Vue - Official (Volar)
6. Tailwind CSS IntelliSense
7. Live Server
```

## Tips Produktivitas

| Shortcut | Fungsi |
|---|---|
| `Ctrl+P` | Quick Open file |
| `Ctrl+Shift+P` | Command Palette |
| `Ctrl+/` | Toggle comment |
| `Alt+↑/↓` | Pindahkan baris |
| `Ctrl+D` | Select next occurrence |
| `Ctrl+Shift+K` | Hapus baris |

## Latihan

1. Install VS Code dan semua extension di atas.
2. Buka folder `latihan-terminal` dari latihan sebelumnya menggunakan VS Code.
3. Aktifkan **Auto Save** dan **Format on Save** di settings.
